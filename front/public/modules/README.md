## 使用说明

- 所有模型文件采用 GLB 格式，已优化压缩，适合网页展示
- 模型支持旋转、缩放等交互操作
- 缩略图采用 PNG 格式，已优化大小
- 模型加载可能需要一定时间，请耐心等待

## 注意事项

- 模型文件较大，建议使用 CDN 加速
- 首次加载可能需要预加载
- 建议在移动端使用低精度模型
